### Фаза 2: Тестування локально (опційно)

Якщо хочеш протестувати на своїй машині перед VPS:

```bash
# 1. Завантажи файли з AI
unzip telegram-moderation-bot-main.zip
cd telegram-moderation-bot

# 2. Встанови залежності
python3 -m venv venv
source venv/bin/activate  # або venv\Scripts\activate на Windows
pip install -r requirements.txt

# 3. Запусти тести
pytest tests/

# 4. Запусти бота локально (з фейковим токеном для перевірки)
# ... або прочитай README для більшого контексту
```

## 🛠 Якщо щось пішло не так

### AI генерує помилковий код
**Рішення:** Передай AI такий текст:
```
"Перепроверь код для помилок. 
Особливо:
- Асинхрона логіка (async/await)
- Обробка помилок
- Типізація (type hints)
Виправ і перегенеруй весь проект."
```

### На VPS не встановлюється python-telegram-bot
**Рішення:**
```bash
# Встанови build-tools
apt install -y python3-dev libffi-dev

# Повтори pip install
pip install -r requirements.txt
```

### Бот не з'єднується до Telegram
**Рішення:**
```bash
# Перевір токен в .env
cat .env | grep TELEGRAM_BOT_TOKEN

# Перевір інтернет-з'єднання
curl -I https://api.telegram.org

# Перезапусти бота
sudo systemctl restart telegram-bot
```

## ❓ FAQ

**Q: Яка мова коду буде?**  
A: Python 3.11+. Это стандарт для Telegram ботів.

**Q: Чи можна запустити без VPS?**  
A: Так, локально для тестування. Але на VPS буде більш стабільно (24/7 без перебоїв).

**Q: Скільки коштує?**  
A: VPS коштує ~5-10$ на місяць. Бот буде на ньому займати <1% процесора.

**Q: Чи це безпечно?**  
A: Так. Конфіг і ключі в .env файлі, база даних локальна, нічого не йде на зовнішні сервери.

**Q: Чи можна розширити функціональність?**  
A: Так! Код написаний модульно. Додавай нові детектори/дії як потрібно.

**Q: Скільки часу займе розробка?**  
A: 
- AI розробляє: 10-30 хвилин
- Встановлення на VPS: 15 хвилин
- Налаштування: 30 хвилин
- **Всього: ~1 година**
